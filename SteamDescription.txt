Multiplier for experience and potential gain.

[h3]How to configure[/h3]
The configuration file will be generated after the game is launched.
Close the game and edit the settings file, which is located at:
[i]<SteamLibraryPath>/steamapps/common/Elin/BepInEx/config/hvits3rk.experiencemultiplier.cfg[/i]

By default, the multiplier is set to [b]2x[/b].
